# Dönem Sonu Projesi

Bu Python projesi, mahalle verileriyle çalışarak kullanıcıya dosya işlemleri, analiz, görselleştirme ve veri güncelleme gibi işlevler sunar.

## 📁 Dosya Yapısı

- `main.py`: Ana program dosyası
- `adding.py`, `deleting.py`, `updating.py`: Kayıt işlemleri
- `file_loader.py`: Veri okuma/yükleme
- `graphics.py`: Grafik oluşturma
- `search.py`, `listing.py`, `analyzing.py`: Arama ve analiz modülleri

## 🛠 Gereksinimler

- Python 3.x
- matplotlib (grafikler için)
